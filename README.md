# ONE桌遊營運管理儀表板

## 專案資訊

- **建立日期**: 2026-03-22
- **用途**: 追蹤年度/月度目標、店家狀態、銷售漏斗、待辦事項
- **技術**: Google Sheets + HTML + Chart.js

## Google Sheets 結構

### 工作表 1：目標追蹤
追蹤年度和月度的各項目標達成狀況

### 工作表 2：潛在客戶
銷售漏斗管理（新詢問 → 有意願 → 接近成交 → 已簽約）

### 工作表 3：已簽約店家
已簽約店家的籌備進度和營運狀況

### 工作表 4：場地開發
新場地開發追蹤

### 工作表 5：待辦事項
交辦任務管理

## 使用方式

1. 更新 Google Sheets 資料
2. 儀表板自動更新（每次重新整理網頁）
3. 手機、電腦都可以看
